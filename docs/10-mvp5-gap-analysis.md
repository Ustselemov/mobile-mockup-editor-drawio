# MVP5 Gap Analysis

## PASS
- infinite board, pan/zoom, minimap, grid/guides
- visual palette with categories and recent/favorites
- mouse-first editing core
- draw.io controlled subset with fallback/warnings

## PARTIAL
- universal artifact coverage
- desktop and generic frames
- connector depth
- inspector redesign for universal artifacts
- export breadth

## FAIL
- multi-page model
- SVG export
- PNG export
- dedicated tree / graph / swimlane layout engines

## Keep
- SVG-first renderer
- store/history/import-export funnel
- board interaction engine
- draw.io compatibility layer

## Refactor next
- document model toward pages + universal node taxonomy
- connector model
- export-image layer
