# V5 Layout And Connector Model

## Current layout modes
- `absolute`
- `vstack`
- `hstack`
- `grid`

## Current connector model
- edge object with source / target ids
- straight or orthogonal routing flag
- mouse-created connector flow
- parent-aware edge ownership

## Planned next layout expansion
- `swimlane`
- `tree`
- `graph`

These are not fully implemented in the current V5 pass and remain tracked as follow-up work.
