# Before / After Plan

## Before
- Text-heavy shell.
- Mixed purpose left panel.
- Layers competed with insertion flow.
- Weak discoverability for contextual actions.
- Empty board felt dead.

## After
- Visual library on the left with clear mode separation.
- Layers moved to a secondary right-side role.
- Board overlays added: quick insert, context toolbar, minimap, empty state.
- Screens demoted to a dedicated mode instead of the main path.

## Next pass
