# Roadmap

This project was generated with the assistance of Codex AI and prompted by Ustselemov.

## Next priorities

1. Extend the layout engine with `grid`, ordering semantics, and stronger container constraints.
2. Implement the MVP3 DSL pipeline: parser, diagnostics, mapper, and editor panel.
3. Add variants and batch screen generation with guardrails and count estimation.
4. Expand template packs for the top-10 product domains from the MVP3 specification.
5. Harden Draw.io import/export coverage for the broader MVP3 subset.
6. Add richer diagnostics, validation reporting, and larger-scale performance checks.

## Quality priorities

- keep documentation in sync with the implemented scope
- expand E2E coverage for palette placement and import/export flows
- add visual regression coverage once the UI structure stabilizes
- reduce the production bundle size warning caused by client-side XML tooling
